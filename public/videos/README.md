# Marine hero background video

Place a seamless loop at:

`marine-hero-loop.mp4`

## Recommended specs

- Duration: 8–12 seconds (seamless loop)
- Aspect ratio: 16:9
- Resolution: 1920×1080 minimum; 3840×2160 preferred
- Codec: H.264 (MP4), web-optimized
- Audio: none (muted background)
- File size: under ~8 MB when possible (use HandBrake or similar)

## Creative direction

- Modern cargo vessel on calm ocean, early golden hour
- Slow cinematic drone: gentle left-to-right pan with slight forward push
- Deep ocean blue, navy, soft grey, natural sunlight highlights
- Open composition on the left for text overlay
- Avoid storms, rough seas, military/cruise ships, fast cuts
